# نظام إدارة المعاهد والدورات التدريبية

نظام رقمي متكامل يهدف إلى تطوير تجربة إدارة المعاهد التعليمية ومراكز التدريب، من خلال أدوات ذكية تسهّل تنظيم الدورات، تسجيل المتدربين، متابعة الأداء، وإصدار التقارير.

## 🎯 المميزات الرئيسية

- إدارة بيانات المعهد والمدربين والمتدربين
- تنظيم وجدولة الدورات والمحاضرات
- طباعة كشوفات ذكية للتحضير
- تقارير جاهزة وقابلة للتخصيص
- دعم تعدد المستخدمين والصلاحيات
- واجهة استخدام عربية سهلة ومتجاوبة
- إمكانية تصدير البيانات
- نسخ احتياطية منتظمة لحماية السجلات
- تكامل مع التطبيقات: الاتصال الهاتفي، واتساب ويب، المشاركة السحابية

## 🧠 التقنية المستخدمة

يعتمد النظام على Microsoft Access مما يمنح مرونة عالية في تعديل التقارير والوصول المحلي أو عبر الشبكة (LAN). لا حاجة لمبرمج خارجي.

## 🧪 نماذج التشغيل

- **نسخة تجريبية مجانية** لمدة 14 يومًا
- **نسخة كاملة قابلة للتفعيل** بعد الشراء
- الميزات المتقدمة غير مفعّلة افتراضيًا (التفعيل اختياري)

## ⚙️ متطلبات التشغيل

- نظام تشغيل يدعم اللغة العربية (عرض من اليمين لليسار)
- إصدار حديث من Microsoft Access يدعم الواجهات الثنائية اللغة
- خطوط تدعم الحروف العربية بوضوح
- إعدادات إقليمية صحيحة (التاريخ، الوقت، الأرقام)
